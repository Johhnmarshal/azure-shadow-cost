# workbooks/

Azure Workbook templates (JSON) generated by the web app for offline use in
the Azure Portal.

**Status:** populated by **PR5** (Context enricher + audit Policy + Workbooks).

Planned templates:

| File | Purpose |
|---|---|
| `azshc-hidden-waste.json` | Hidden-waste leaderboard, per-category bars, top-25 offenders, sub leaderboard. |
| `azshc-peak-rightsizing.json` | VM peak metric heatmaps + Advisor-unsafe filter. |
| `azshc-ri-coverage.json` | Coverage gap by family×region + cumulative buffer-exposure tracker. |

To use a template:

1. Download the JSON from the web app's Downloads tab (post-PR5).
2. Azure Portal → **Monitor → Workbooks → New → Advanced editor → Gallery Template**.
3. Paste the JSON and Save.
