What this app does (and what Advisor doesn't)

Azure Advisor is excellent at the utilization story — idle VMs, low-use disks, RI suggestions. It is largely silent on the classical FinOps shadow-cost surface:

  • Allocation gap — resources missing required tags; the Visibility Gap %.
  • Commitment drift — RIs / SPs you bought that are now under-utilised.
  • Data-plane waste — over-redundant storage, long Log Analytics retention, multi-region Cosmos in non-prod.
  • PaaS sprawl — empty App Service Plans, idle workspaces, dormant APIM.

For each detector we surface a finding card with the full FinOps ROI-engine contract — recommendation, $/month, eng hours, risk, tier, and a business-value framing — and we generate a safe bash script (dry-run by default) you can review and apply.

Detectors in this build

  • unattached managed disks · unused Standard public IPs · empty App Service Plans
  • tagging gap (per resource type)
  • commitment drift (Reservations / Savings Plans, last 30 days)
  • over-redundant non-prod storage · long Log Analytics retention · multi-region non-prod Cosmos
